# Scriptura

Prototype de site biblique immersif en style ancien parchemin avec animations 3D.

## Lancer localement

Ouvrir `index.html` dans un navigateur moderne, ou servir le dossier avec un petit serveur statique:

```bash
python3 -m http.server 4173
```

Puis aller sur `http://localhost:4173`.

La scene 3D utilise Three.js via CDN.
